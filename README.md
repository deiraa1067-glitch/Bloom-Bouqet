# 🌸 Bloom Bouquet
### E-Commerce Website for Flower Bouquet Shop

## 📖 Business Overview

### 📌 Business Name
**Bloom Bouquet**

### 📌 Business Description
Bloom Bouquet adalah website e-commerce yang menjual berbagai jenis buket bunga berkualitas tinggi untuk berbagai acara spesial seperti wisuda, ulang tahun, anniversary, Hari Valentine, pernikahan, dan hadiah untuk orang tersayang. Website ini dirancang agar pelanggan dapat dengan mudah melihat katalog produk, menambahkan produk ke keranjang, melakukan checkout, dan memilih metode pembayaran secara online.

### 🌟 Value Proposition
Bloom Bouquet menawarkan pengalaman berbelanja buket bunga yang mudah, cepat, dan nyaman dengan keunggulan:

- 💐 Desain buket modern dan elegan
- 🌹 Bunga berkualitas premium
- 🎁 Tersedia custom bouquet sesuai permintaan
- 🚚 Proses pemesanan mudah
- 💳 Simulasi pembayaran online
- 📱 Tampilan responsif untuk desktop dan mobile
